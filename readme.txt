== Description ==
Create simple users register form in your site.
User can fill up there details and registered using simple register user form.

== Installation ==
1. Download plugin and unzip it to your `/wp-content/plugins/` plugin directory.
2. After that go to your plugin manager and activate it.
3. After activate simple users register form one configuration link is show in plugin manager and it is also show in admin menu.
4. Go to simple users register form and manage form inputs that are you want to display.
